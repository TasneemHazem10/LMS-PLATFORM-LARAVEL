# LMS Platform - Learning Management System

A full-featured Learning Management System (LMS) built with Laravel, MySQL, and modern HTML/CSS/JavaScript. This platform allows users to register, browse courses, enroll, watch lessons, take quizzes, and track their learning progress.

## Features

### User Roles
- **Admin**: Manage users, categories, courses, lessons, and view enrollments
- **Student**: Browse courses, enroll, watch lessons, take quizzes, track progress

### Core Features
- User registration with role selection (Student/Instructor)
- Role-based authentication and authorization
- Course management (create, edit, publish)
- Lesson management with video support
- Quiz system for each lesson
- Progress tracking and completion certificates
- Category-based course organization
- Featured courses
- Responsive design

### Pages
- **Public**: Home, Course Catalog, Course Details, Login, Register
- **Student**: Dashboard, My Courses, Course Player, Lesson View
- **Admin**: Dashboard, Users, Categories, Courses, Lessons, Enrollments


### Prerequisites
- PHP 
- Composer
- Node.js & npm
- MySQL or SQLite
